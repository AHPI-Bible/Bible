# AHPI 성경 주해 프로젝트

Sefaria API와 AHPI 자체 데이터베이스를 결합하여 만든 하이브리드 성경 웹 애플리케이션입니다.

## 주요 기능
- **좌우 대조:** 한글(개역한글), 영어, 히브리어/헬라어 본문을 동시에 볼 수 있습니다.
- **단어 분석:** 히브리어 원문 단어를 클릭하면 Sefaria 사전 데이터를 통해 뜻을 팝업으로 보여줍니다.
- **AHPI 주석:** 각 절에 대한 독자적인 주석을 제공합니다.

## 기술 스택
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Data Source: Sefaria API (Public), Local CSV (Korean Text)
